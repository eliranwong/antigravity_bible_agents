# Locations Retrieval Skill

## Overview
This skill allows the agent to retrieve geographical coordinates and historical-etymological descriptions of locations mentioned in the Bible. It queries a local coordinates dictionary and retrieves details from the local `exlb3.data` SQLite database, cleaning the HTML content for display.

## Guidelines & Objectives
When executing this skill:
- Always run the python retriever script located at `.agents/skills/locations/locations_query.py` to retrieve location data.
- Execute the script using: `python3 .agents/skills/locations/locations_query.py "<query>"` where `<query>` is the name of the location.
- The script automatically handles fuzzy spelling matching (Levenshtein distance) for locations with minor spelling variations.
- Present the exact output of the script to the user. The script filters out embedded HTML map elements but preserves the Live Google Map link, coordinate details, dictionary descriptions, and occurrences in scripture.
- All study outputs generated by the `/locations` command must be saved to the `biblemate/` folder with a timestamp prefix under the Universal Study Output Saving Rule.
