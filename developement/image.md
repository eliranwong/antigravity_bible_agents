Create a new skill `image` and slash command `/image` to generate bible-related images.

Generated images are placed in the `images/` directory. Remember not to hardcode absolute path to make this repository portable.

Each filename begins with a timestamp and the image title, e.g. with a filename format of 'YYYY-MM-DD-HH-MM-SS_<image-title>.png' for the date and time of the image generation.
